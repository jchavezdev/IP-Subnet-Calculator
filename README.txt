Instalar python 3.x

1.-Visitar el sitio oficial de python, la página detecta automáticamente si se usa windows desplegando 
una lista de ejecutables compatibles
2.-Elegir descargar python 3, puede ser la versión más actual
3.-Ejecuta el archivo después de descargarlo
4.-Marca la casilla "Añadir Python 3.5 al PATH"
5.-Haz click en "Instalar ahora"
Ya esta instalado python

Ejecutar un programa python

1.-Abrir símbolos del sistema, cdm
2.-Ubicarse en la carpeta donde está el script que queremos ejecutar
3.-Escribir "python" y luego el nombre del script sin olvidar la extensión de este script
Ya se ejecuta python

Usar el proyecto

El programa permite al usuario obtener los datos para configurar una red, los datos el 
resultado es desplegado en la línea de comandos y los datos introducidos en una ventana programada
